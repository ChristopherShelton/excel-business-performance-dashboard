# Excel Business Performance Dashboard

This project showcases an Excel-based business intelligence dashboard analyzing retail sales performance across time, product categories, geography, customer demographics, and sales channels.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Metrics
- Slicers

## Key Metrics
- Total Sales
- Total Transactions
- Average Order Value (AOV)
- Total Units Sold
- Average Customer Age

## Dashboard Features
- Sales trends over time
- Sales by product category
- Top 5 cities by sales
- Sales by state
- Sales by channel (Online vs In-Store)
- Customer age distribution
- Interactive slicers for:
  - Product Category
  - Region
  - Channel

## Files in This Repository
- `excel-business-performance-dashboard.xlsx`  
  Main Excel file containing the dataset, pivot tables, KPIs, and dashboard.

- `dashboard-preview.jpg`  
  Screenshot of the completed dashboard.

## Summary
This project demonstrates practical Excel skills commonly used in data analyst roles, including data aggregation, KPI development, dashboard design, and interactive filtering for business insights.
